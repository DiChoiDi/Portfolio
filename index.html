Projects

UFO - Software Lead

August 2011 - May 2012

Project UFO (Unmanned Fighting Object) is a tank-like product controlled through a wireless network. It was designed and developed by a team of five, including myself, for the CSE Senior Design course at UTA. The software was built on .NET 4.0 for the Base Station and Mono .NET for the Remote Station (vehicle). The vehicle was hand built and equiped with a Mini ITX board running the Ubuntu OS. Mono was used to do cross-platform .NET development for code consistency and maintenance. The requirements involved providing video visuals for the user at the base station and basic tank functionality (movement and firing). Overall the project was a complete success, completing two weeks before the deadline.
